# Plots

The `DataPlots` class provides interactive visualizations for financial returns data.
Access it via the `.plots` property on a [`Data`](data.md) object.

::: jquantstats._plots.DataPlots

The `PortfolioPlots` class provides interactive visualizations for portfolio data.
Access it via the `.plots` property on a [`Portfolio`](portfolio.md) object.

::: jquantstats._plots.PortfolioPlots
